# 🌾 KrishiConnect

KrishiConnect is an AI-powered smart agriculture platform built using Streamlit.  
The platform helps farmers access market prices, weather updates, crop insights, government schemes, profit analysis, and AI-based predictions in one place.

---

# 🚀 Features

## 📊 Dashboard
- Interactive farmer dashboard
- Agricultural-themed UI
- Quick analytics overview

## 💹 Market Prices
- Live crop price viewing
- Market comparison
- Crop ranking system

## 📈 Price Trends
- Crop price trend visualization
- Charts and analytics

## ☁ Weather Updates
- Weather information dashboard
- Farming condition support

## 🤖 AI Prediction Bot
- AI-based farming assistance
- Smart recommendations

## 💰 Profit Calculator
- Estimate farming profit
- Expense and income analysis

## 🛒 Cart & Wishlist
- Add agricultural products
- Manage wishlist and selections

## 📰 News & Alerts
- Agricultural news updates
- Important farmer alerts

## 🏛 Government Schemes
- Farmer welfare schemes
- Agriculture support programs

# ▶ How to Run Locally

## Clone Repository

```bash
git clone https://github.com/gulnaazaramakandarmakandar-gif/krishiconnect.git
```

## Install Dependencies

```bash
pip install -r requirements.txt
```

## Run Streamlit App

```bash
streamlit run app.py
```
